# LeetCode - Easy Problems

This folder contains solutions to problems classified as "Easy" on LeetCode.

## Structure

- Each subfolder is named with the problem number and name (e.g., `1-two-sum/`).
- Each problem folder contains at least a `solution.py` file.

## Requirements

- Python 3.x installed

## Usage

1. Navigate to the desired problem folder:
	```bash
	cd 1-two-sum
	```
2. Run the solution file:
	```bash
	python solution.py
	```
